# CLMS - Clinic Management System

CLMS (Clinic Management System) is a comprehensive, responsive web application built with **PHP** and **MySQL** to optimize, manage, and digitize clinical lab workflows. It bridges communication gaps between administrators, doctors, and medical laboratory technicians, enabling efficient tracking of patient diagnoses from prescription to final invoice billing.

---

## 🚀 Key Features

### 💻 Admin Dashboard
- **Comprehensive Analytics:** Track total daily appointments, active test orders, registered doctors, and pending laboratory test modules.
- **Staff Management:** Securely add, edit, or remove Doctors and Medical Laboratory Technicians (MLT).
- **Test Modules configuration:** Define custom clinical tests, cost brackets, and drug inventories dynamically.

### 🩺 Doctor Panel
- **Patient Portals:** Direct access to lookup specific patient medical histories.
- **Prescription Manager:** Instantly generate electronic scripts detailing required laboratory checks and medications.
- **Status Tracking:** Review previous appointments or check whether prescribed diagnostic orders are compiled or pending.

### 🧪 Technician Workflow & Automated Billing
- **Interactive Schedules:** View assigned testing tasks based on operational real-time queues.
- **Billing Automation:** Generate detailed laboratory summary slips itemizing diagnostic test expenses.
- **PDF Export Capabilities:** Integrated with `dompdf` to instantly produce downloadable, clean clinical invoices for patients.

---

## 🛠️ Tech Stack

- **Backend:** PHP (Object-Oriented approaches with standard Session states)
- **Database:** MySQL / MariaDB
- **Frontend Framework:** HTML5, CSS3, SCSS, JavaScript (ES6+)
- **UI Architecture:** Template architecture inspired by SB Admin 2 (Bootstrap 4 compilation)
- **Dependencies:** `dompdf` package for dynamic automated system document rendering
